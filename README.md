# AI_Budget-Mate
> **"우리들의 소비를 영(Young)리하게 돕는 AI 파트너"**

## 📌 프로젝트 개요
대학생이라면 꼭 챙겨야할 편의점 혜택들을 AI Budget Mate가 놓치지 않도록 도와드립니다 !

🙋🏻‍♀️ **나는 어떤 소비 스타일을 가지고 있고, 편의점에서 얼마나 많이 소비를 할까?**

🙋🏻‍♂️ **"여기 통신사 할인이 되나?", "이 카드가 여기서 혜택이 있었나?"**

🙋🏻 **"내가 주로 가는 GS25나 CU 편의점, 어디서 더 많이 아낄 수 있을까?"**

고민하다 그냥 결제하진 않으셨나요?
이 모든 고민들 AI_Budget-Mate에게 맡기세요 ! 


## 🎯 주요 타겟 및 해결 과제
- **타겟**: 복합적인 카드 혜택과 통신사 등급을 보유한 대학생


## 🛠 주요 기능 
### 1️⃣ 실시간 행사 상품 매칭 
내 주변 편의점(CU, GS25 등)의 1+1, 2+1, 월별 할인 행사 데이터를 실시간으로 조회 ! 

### 2️⃣ 하이브리드 할인 계산기 
* **통신사 할인**: 내 멤버십 등급(VIP/Gold/Silver)에 따른 할인액 계산
* **카드 혜택**: 내가 보유한 카드중 편의점 업종별 최대 할인/적립/캐시백 혜택을 가진 카드 추천


## 🛠 기술 스택 
- **Database**: PostgreSQL (JSONB 타입을 활용한 유연한 혜택 저장)
- **AI/Logic**: Python (Scikit-learn을 활용한 소비 패턴 분류 및 최적화 추천)
- **Data**: Selenium/BeautifulSoup (카드사/통신사 혜택 정보 자동 수집)
